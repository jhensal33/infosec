How to connect to the database:

1. Download Software SQL server Management Studio(SSMS)
https://docs.microsoft.com/en-us/sql/ssms/download-sql-server-management-studio-ssms?view=sql-server-ver15

2. click the "connect object explorer" on the right side of "Connect" in the object explorer window (a plug shape)

3. In the "Server name" box, enter your laptop's IP number along with a comma and a 1433.
	e.g. if my laptop's IP address is 192.168.56.1, then the Server name should be "192.168.56.1,1433"

4. In the "Authentication" box select SQL Server Authentication.

5. Typed Login: admin	password: 4471

6. Click connected

7. Should connected to our database.
